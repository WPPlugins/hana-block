A simple and powerful plugin that allows you to create content block and display anywhere.

== Description ==

<a href="http://rewindcreation.com/hana-block">Hana Block</a> is a very simple and yet powerful plugin.

* Custom post type `hana_block` is created to manage content block using **use the WYSIWYG editor**
* There are 2 types of content blocks: Layout and Content.
* The content block can be included in post, page and text widget using short code `[content-block]`
* The content block can be included in any theme supported action hooks.

Support is provided at <a href="http://rewindcreation.com/support">support forums</a>.

== Installation ==

1. Unzip the `hana-block.zip`.
1. Upload the `hana-block` folder to your `/wp-content/plugins` directory.
1. Activate *Hana Block* in Plugin section

== Frequently Asked Questions ==

= Why was this plugin created? =

One of the WordPress' weakness is lack of content blocks. The plugin is created to allow users to maintain and use the repeatable contents.
